SOBRE MIM

Desde muito pequeno sempre gostei da área de tecnologia, mas só no começo de 2023, decidi que era na área de programação que gostaria de trabalhar, desde então, venho aprendendo, aprimorando minhas habilidades, e sempre tentando fazer projetos cada vez mais complexos, usando bibliotecas, API´s ou frameworks que acabei de aprender ou conhecer. Atualmente estou focando na área voltada ao Front-End, porém pretendo no futuro aprender também sobre Back-End. Atualmente estou em busca da minha primeira oportunidade profissional.

MINHAS HABILIDADES

HTML5

HTML5 é uma linguagem de marcação de texto, responsável por toda a estuturação do site como: cabeçalhos, formulários, menus, ETC...

CSS3

CSS3 se trata de uma linguagem de estilização, responsável por estilizar o elementos HTML da aplicação

JavaScript

JavaScript é a linguagem de programação que permite dar interatividade ás aplicações, também é base de muitas bibliotecas e frameworks.

React JS

React js é uma biblioteca javascript, na qual serve para componentizar os elementos HTML, asim, tornando mais fácil a manutenção dos sites/aplicações Web.

Tailwind CSS

Tailwind CSS é uma bliblioteca CSS, que permite estilizar os elementos HTML por meio de classes pré estabelecidas.

Git

O git é uma ferramenta de versionamento de código, ela facilita o gerenciamento e o compartilhamento dos códigos de uma aplicação entre desenvolvedores

GitHub

Com o GitHub, os desenvolvedores são capazes de compartilhar os códigos para outros desenvolvedores, parecido com redes sociais.

Responsividade

É fundamental que as aplicações web sejam acessíveis para todos os tamanhos de tela, tanto para computadores e laptops, tablets e smartphones.
